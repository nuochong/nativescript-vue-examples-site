<Page>
  <ActionBar title="My App" />
  <GridLayout>
    <Label text="My Content"/>
  </GridLayout>
</Page>

# 道具

名称 类型 描述
actionBarHidden Boolean 显示或隐藏ActionBar。
默认值： false
backgroundSpanUnderStatusBar Boolean 获取或设置页面背景是否跨越状态栏。
默认值： false。
androidStatusBarBackground Color 获取或设置android上状态栏的颜色。
enableSwipeBackNavigation Boolean 获取或设置是否可以在ios上刷回页面。
默认值： true。
statusBarStyle String 获取或设置状态栏的样式。
有效值：
"light",
"dark"

# 活动

名称 描述
navigatedFrom 导航父框架时发出。
navigatedTo 父框架导航到此页面时发出。
navigatingFrom 父框架导航离开此页面时发出。
navigatingTo 父框架导航到此页面时发出。
